# Project Instructions

This is a simple personal portfolio site built with React and Vite.

## Goal
Create a clean one-page scrolling personal site with:
- top navigation bar
- About section
- Projects section
- Links/Contact section

## Constraints
- Keep the first version simple.
- Do not add routing or multiple pages.
- Do not add complex animations yet.
- Do not use external UI libraries unless asked.
- Prefer plain CSS in `src/App.css`.
- Use simple React components only if they make the code clearer.

## Style direction
Start unstyled or minimally styled. Structure and content come first.


## React
Use clean, simple React components.

Prefer a small number of readable components:

- Navbar
- Section
- ProjectCard
- Links


Do not over-engineer the structure.

Keep components in `App.jsx` for now unless the file becomes messy.
